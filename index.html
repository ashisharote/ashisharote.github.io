About Me

Currently, I am working as a Assistant Professor at NEXTA (Next Generation Tatara Co-creation Center), Shimane University, JAPAN. Moreover, I am working on the microscopic solidification of super-alloys. This study is carried out with the numerical simulations using in-house codes. Primary objective of my study is to identify the fundamental physics of dendrite formations under specific solidification conditions.

I completed my PhD in Mechanical engineering from National Institute of Technology, Surat. During this period, I worked on the numerical simulations of oscillating liquid jets, and have tried to investigate their fundamental properties. Moreover, during my masters program I had also worked on the development of the CRDI engine with programmable ECU, and investigated the role of fuel injection pressure, piston geometry and EGR rates on engine performance.

My research interests are as follows,
  • Numerical investigations of fluid instabilities
  • Development of the efficient and accurate multiphase flow methods
  • Microscopic solidifications in super-alloys
  • Oscillating/Sweeping liquid jets
  • Cellular automata methods for multiphase simulations

Research

Stability of Oscillating Jets

Stability of Oscillating Jets


Description

The developing oscillating jet and its disintegration at the downstream. The pressure drop across the jet interface causes the increase in the oscillation amplitude. This study was aimed towards understanding the fundamental reason for the jet instability and effect of flow parameters on it.

[Reference]

Coherent Vortical Structures
in Oscillating Jets

Coherent Vortical Structures in Oscillating Jets


Description

Formation and merging of KH rollers behind the jet front (Re = 4500) shown by the numerical results for Q-criterion = 0.05 (white) sliced by z-normal plane with deflected jet (blue). This study was aimed to identify the dominant coherent structures formed around the jet. This study also analysed the air entrainment potential and the effect of viscous forces on it.

[Reference]

Tandem Oscillating Jets

Tandem Oscillating Jets


Description

Asynchronous oscillating tandem jets (blue) with their vortical structures (Q-criterion = 0:05) (white) sliced and an enlarged view of the interaction zone depicting the velocity vectors (red) scaled by factor of 0.5. The study of tandem oscillating jets was aimed to understand the instability when jets are placed in tandem and effect of oscillation phase was analysed.

[Reference]

Development of Accurate and
Efficient Algebraic VOF Method

Development of Accurate and Efficient Algebraic VOF Method


Description

A novel algebraic VOF method is proposed named Smoothly Adapting Interfacial Scheme based on Hybridization (SAISH). SAISH scheme is computationally efficient and accurate method amongst the unsplit algebraic VOF methods available in the literature.

[Reference]

Blended Advection Schemes

Blended Advection Schemes


Description

Comparison of analytical results for a) water height and b) velocity along the domain length with the blended scheme and QUICK at t=0.2 (Sod shock tube problem). The blend of higher order scheme with the Fromm schme leads to lower dispersion errors.

[Reference]

Dendrite Simulations
in Metal Alloys

Dendrite Simulations in Metal Alloys


Description

Columnar dendrite results for Al-4wt%Cu alloy with 8K undercooling and directional cooling of 2000K. These results are based on the Cellular automata method with our newly proposed cell capturing methodology.

Codes

The codes provided below are written by me, and are not by any means optimized versions of the code. Hence, please use them only as a guidline. These codes are written in C++ using finite volume method to solve Navier-Stokes equations and the files are available for download here. If there are any queries regarding the code please feel free to contact me.


  • 2D Lid-driven Cavity problem : code download

    The code uses a blended advection scheme. The NS solver is solved using a semi-explicit solver, where velocity is solved explicitly whereas, pressure is obtained implicitly. The code is made parallel using OpenMP API.

  • Piecewise Linear Interface Calulations (PLIC) resconstruction (casewise)code (Uniform Grid): code download Torsten Sillke's flower
  • Piecewise Linear Interface Calulations (PLIC) resconstruction code (Non-Uniform Grid): code download Torsten Sillke's flower
  • PLIC-VOF (Pure Advection Test Cases) code download

    This code solves the Volume-of-Fluid equation geometrically using PLIC reconstruction. The VOF equation is made second order accurate by using operator-split method.

  • Dam-Break Test Case: code download

    This code solves the Navier-Stokes equations which are coupled with PLIC-VOF solver. The test case is based on Martin et al.'s paper.

  • Rayleigh-Taylor Instability Test Case (Rudman 1997): code download

  • Algebraic VOF (SAISH VOF scheme): code download

    The code solves volume fraction equation using a novel compressive interrface capturing scheme called "Smoothly Adapting Interfacial Scheme based on Hybridization" (SAISH). The code also involves the option to solve the VOF equation using other algebraic VOF methods such as CICSAM, M-CICSAM, CUIBS and FBICS.
